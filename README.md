# AIGO-2023-HW

This is a small project for HomeWork of class Open AI API 之自然語言處理入門與實務
請使用Python程式語言呼叫OpenAI API，
取得某一段英文文稿的中文結論，
若時間許可，請以GUI方式呈現，
若時間很多，請使用OpenAI之pre-trained whisper模型處理。
輸入: 英文文稿 [input.txt]
輸出: 中文結論 [output.txt]


## Requirest python package
